# hydrocarbon-synergistic-guilds

Code and processed data organization for the manuscript:

**“Contrasting hydrogeological settings structure stage-coupled synergistic guilds underlying petroleum hydrocarbon degradation”**

## Overview

This repository contains the analysis scripts used to investigate petroleum hydrocarbon degradation in contaminated soils under contrasting hydrogeological settings. The workflow integrates hydrocarbon fraction measurements, geochemical data, genome-resolved metagenomics, synergy analysis, and stage-coupling analysis.

## Manuscript status

unpublished manuscript

## Main contents

- `scripts/`  
  Analysis and plotting scripts used in the manuscript.

- `figure_inputs/`  
  Processed input files used to generate the main figures.

- `processed_data/`  
  Intermediate analysis outputs, including hydrocarbon-associated MAG tables, module score matrices, and stage-coupling results.

## Reproducibility

The repository is organized around the main analyses reported in the manuscript, including:

1. Identification of hydrocarbon-associated MAGs  
2. Functional module scoring and KO-to-module assignment  
3. Synergistic taxon-pair analysis  
4. Stage-coupling and depth-resolved succession analysis  


Scripts are named to match the corresponding analyses and figures wherever possible.
